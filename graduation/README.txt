Цель работы сравнить метрику даваемую подходом U-NET-2D для задачи детекции с метрикой даваемой Encoder-ом от этой сети с многократными предсказаниями и сдвигами входного изображения на одном классе частиц apo-ferritin.
Был предложен алгоритм пост обработки для обоих подходов, написан код с использованием языка python, tensorflow, numpy и других. 
Это первый мой опыт, выложенны черновики работ.
